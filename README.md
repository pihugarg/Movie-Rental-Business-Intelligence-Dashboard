
# 🎬 Movie Rental Business Intelligence Dashboard

Interactive Power BI dashboard built using multiple relational tables to analyze revenue, customer behaviour, movie performance, and store operations through KPI-driven business insights.

---

## 📌 Project Overview

The **Movie Rental Business Intelligence Dashboard** is an interactive Power BI solution developed using the **Sakila DVD Rental Database**. The project transforms raw transactional data into meaningful business insights through interactive dashboards, KPI cards, DAX calculations, and business recommendations.

The dashboard enables stakeholders to monitor business performance, understand customer behavior, evaluate movie and category performance, and analyze store operations to support data-driven decision-making.

# 💡 Key Business Insights

- Generated over **67K** in total rental revenue.
- Managed **16K** rental transactions across **599** customers.
- **Telegraph Voyage** emerged as the highest revenue-generating movie.
- **Sports** and **Action** categories contributed significantly to overall revenue.
- **PG-13** movies generated the highest share of rental revenue.
- **India** had the highest customer concentration.
- Store-level performance comparison highlighted opportunities for inventory optimization.
- Seasonal rental trends helped identify peak rental periods.
